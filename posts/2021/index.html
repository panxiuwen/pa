Word相关

使用撇号的时候用'，而不是’，有时候会出现显示问题，值得注意
在编辑文档前先设置好图片的分辨率，推荐330
如果文档较大的话，可以把文档后缀加.zip，解压后看看那些文件出了问题
删除全部图片：替换^p为空格
删除作者信息：文件->信息->右键删除
如果导出的pdf缺失部分图像：选项->显示->打印选项->打印在Word中创建的图形

设置Office不压缩图像

选项->高级->不压缩文件中的图像
如果图像过大，建议选择高保真，不影响打印和观看，如果是位图，建议使用png格式，体积又小质量又高jpg格式会出现色块
PS：貌似只针对单个文档生效

XPS分峰添加阴影

方案1

在Plot Details中选中曲线 -> Fill Area Under Curve Enable -> Normal/Base Auto -> Pattern Fill里选Tow Colors
将Background线里的Fill改为白色，可以在线里调transparency

方案2

在Plot Details中选中曲线 -> Fill Area Under Curve Enable -> File to date plot- One Color -> Pattern Fill里选Tow Colors
可以在线里调transparency

方案3(不推荐)

单击选中曲线 -> Gadgets -> Integrate
在弹出的Dialog的Integration勾选Keep the shading color after New Output, Baseline里的Mode选Use Exitsting Dateset，选择好积分位置后点击阴影右上角箭头，选择New Output，后点X关闭
双击阴影，在Dialog的Fill选项的Gradient Fill里选择Two Colors

其他问题

如果出现峰相互遮住的话，双击页面左上角的1，打开PlotSetup，在出现的菜单里上下调整曲线
如果需要自定义刻度内容，在Tick Lable -> Display里设置Type -> Tick-indexed string 再设置String内容，用空格隔开

Endnote使用

谨记：Endnote导入的文献可能不规范，请注意期刊是否简写，或首字母是否大写，导出文献后检查上下标和菌种斜体

文献style文件存放目录

可自行编辑后分析存储
C:\Users\你的用户名\Documents\EndNote\Styles
放在安装目录里也行，一般是
C:\Program Files (x86)\EndNote 20\Styles

插入后显示{Author，#year}

插入文献后，点击一下Bibliography里面的Update Citation and Bibliography

修改杂志的缩写

菜单 -> “Tools” -> “Output Style” -> “Edit” -> “Journal Name” -> “Abbreviation X”
菜单 -> “Library” -> “Open Term List” -> “Journals Term List”-> “Edit Term”
Term文件位置”/PATH/Terms Lists”

修改引用格式

菜单 -> “Tools” -> “Output Style” -> “Edit” -> “Bibliography” -> “Templates”
引文格式位置”/PATH/Style”

修改显示页码

菜单 -> “Tools” -> “Output Style” -> “Edit” -> “Page Number” -> Show the full range of pages

Word引用的Endnote库部分丢失或不全

Word角标->右键->Edit Citation->More->Tools->Export traveling library

ANOVA分析
