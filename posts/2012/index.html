常见离子的颜色与性质

2019年 8月 03日 发布在 Research

本文介绍了常见离子的颜色与性质

TiCl4为无色液体；TiCl3为紫；Ti(OH)3也是紫色沉淀；Ti4+、TiO2+是无色离子；

V2+紫；V3+绿；VO2+蓝；VO2+浅黄或深绿；VO43-无色；V2O5红

Cr2O72-橙；CrO42-黄；Cr3+绿；Cr(OH)3灰蓝；溶于NaOH得[Cr(OH)4]-绿，不溶于氨水；PbCrO4黄；BaCrO4黄；K2CrO4黄；CrO5蓝；[Cr(H2O)6]3+紫；[Cr(NH3)6]3+黄；

MnO2棕黑；Mn(OH)2白，不溶于NaOH在空气中氧化MnO(OH)白变棕；MnS粉；Mn2+=[Mn(H2O)6]2+浅红；MnO42-绿；MnO4-紫；

Fe(OH)3棕；[Fe(H2O)6]3+淡紫；[FeF5]2-无；[FeO4]2-紫红；[FeCl6]2-黄；[Fe(SCN)6]3-血红色；[Fe(CN)6]4-黄；[Fe(CN)6]3-深红；KFe(CN)6蓝；

Co2+=[Co(H2O)6]2+粉；无水CoCl2为蓝色；Co(OH)2粉；[CoCl4]2-蓝[Co(NH3)6]2+黄；鉴别：与KSCN作用后乙醚层为蓝色

Ni2+绿； Ni(OH)2绿不溶NaOH溶于氨水；Ni(OH)3黑；[Ni(NH3)6]2+蓝鉴别：与丁二酮肟作用鲜红色；

Cu2O砖红；CuX白；Cu(CN)43-无；Cu(NH3)2+无；CuO黑；CuS黑；Cu2+=[Cu(H2O)4]2+蓝；[Cu(OH-)4]2-蓝；[CuCl4]2-黄；[Cu(NH3)4]2+深蓝；[Cu(CN)4]2-无；

ZnS白；Zn(OH)2白；Zn(OH)42-无；[Zn(NH3)4]2+无；

Sn(OH)2白；Sn(OH)42-无；SnS2黄；SnS32-无；SnCl4无色液体；Sn2+与H2S生成SnS棕溶于Na2S

PbO黄；Pb3O4红；PbO2棕黑；PbS黑；Pb3O4白；Pb(OH)2白溶于NaOH生成Pb(OH)42-无被ClO-氧化成PbO2棕黑；Pb2+与盐酸生成PbCl2白；盐酸过量溶解PbCl42-(未知)；与碘生成PbI2黄；和重铬酸根生成PbCrO4黄

Ca和H2PO4-不沉淀，继续加碱生成CaHPO4沉淀

As2O3白在水中溶解度较小但溶于盐酸生成AsCl3无色液体，溶于NaOH生成AsS32-

As2S3黄溶于KOH和H2O2的混合溶液生成AsS32-As2S5黄；

SbCl5能存在溶液与H2S生成Sb2S3橙；Sb2S5橙；

BiCl3白溶于水部分水解为BiOCl沉淀与H2S生成Bi2S3棕黑溶于盐酸不溶于氢氧化钠

BiCl3与NaOH反应Bi(OH)3白色不溶氢氧化钠

BiCl3与Sn(OH)42-(SnCl2+NaOH)还原成黑色固体Bi还原为黑色物质

Ag2S黑；AgNO2浅黄；AgN3白；

Ag2CrO4砖红；Ag3PO4黄；Ag3AsO4红棕；Ag2(S2O3)白；

CdS黄；CdO棕；CdI2黄；Cd(NH3)2+无；

Hg2Cl2白；Hg2I2黄；HgO黄(红)；HgS黑(红)；HgI2红；Hg2NI·H2O红；[HgI4]2-无色；HgO·Hg(NO3)2黄；HgNH2Cl白；

ZnS不溶氢氧化钠但Zn(OH)2溶于NaOH,NH3

HgCl2溶于水

Mg3N2白

As3+和ClO-反应生成AsO3-，酸性条件氧化

As2S3 + H2O2 + OH- = AsO3-

Ag3N + H2S = Ag2S + (NH4)2S + N2

2[HgI4]2+ + 12OH- + NH4+ = Hg2NI·H2O + 7I- + 7H2O

AsO3- + Mg(NO3)2 + NH4NO3 = MgNH4 + AsO3白色

溶于NaOH不溶于氨水Cr(OH)3、Sn(OH)2、Pb(OH)2

不溶NaOH溶于氨水Ni(OH)2,Co(OH)2

不溶NaOH不溶于氨水HgO、Fe 、Mn(OH)2

既溶NaOH也溶于氨水Cu(OH)2、Zn(OH)2

不溶于盐酸溶于氢氧化钠As2S3黄

溶于盐酸不溶于氢氧化钠Bi2S3棕黑
